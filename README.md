# CoverIQ Studio - Cover Art A/B Tester & Analyzer

Professional, client-ready web application for testing song and album cover art across major streaming platforms.

## Highlights

- Upload cover art via drag-and-drop, file picker, or URL.
- Compare artwork across 11 platform UI replicas with synchronized device, theme, and player state modes.
- Inspect artwork in a dedicated single-song view with palette extraction and contrast cues.
- Generate AI-style heuristic insights, scores, and batch-testing leaderboards.
- Export-ready report and batch export placeholders for production workflows.

## Getting Started

This project is fully client-side.

```bash
python -m http.server 8000
```

Then open `http://127.0.0.1:8000/index.html`.
